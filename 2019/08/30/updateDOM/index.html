修改 DOM 元素的方法

  • 发布 2019-08-30
  • 更新 2019-08-31
  • 阅读

修改内容（3种方法）

  1. elem.innerHTML: 获取或设置元素开始标签到结束标签之间的原始 HTML 代码片段；

  2. elem.textContent: 获取或设置元素开始标签到结束标签之间的纯文本内容（要考虑兼容性问题 IE9+）；

  3. 获取表单元素的内容：value

修改属性（2种方法）

  1. 核心 DOM：操作一切结构化文档的 API(HTML, XML)：
    elem.getAttribute(); elem.setAttribute(attr, …);
    判断是否包含属性：elem.hasAttribute();
    移除属性：elem.removeAttribute();

  2. HTML DOM：对部分常用DOM的简化版本（常用）：
    获取属性值：elem.属性名；
    修改属性值：elem.属性名 = ‘值’；
    判断是否包含属性：elem.属性名 === ‘’不包含；
    移除属性：elem.属性名 = ‘’；

延伸 – 三大状态属性

我们常在表单元素中遇到三大状态属性：selected checked disabled；

    后续再更改网站样式的时候，如果样式比较多，首选使用类，然后把类添加到内容里面；这样便于代码的维护性。

  2. 添加和删除：

    (1) 末尾加 document.body.appendChild(Jay); // 需要添加到body中；

    (3) 替换：parent.replaceChild(child, oldChild);

优化：

尽量少的修改DOM树
原因：页面加载过程中，尽量少的修改DOM，每次修改，都会导致DOM树的重排和重绘，这样会对性能造成影响；

render Tree（圣诞树）– 每次修改DOM树，都会导致DOM树的重新计算高度和宽度；
